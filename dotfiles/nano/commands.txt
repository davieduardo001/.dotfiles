1. **Salvar e Sair:**
   - `Ctrl + O`: Salvar o arquivo.
   - `Ctrl + X`: Sair do Nano (pode pedir para salvar se as alterações foram feitas).

2. **Navegação:**
   - Setas direcionais: Mover o cursor pelo texto.
   - `Ctrl + F`: Avançar uma página.
   - `Ctrl + B`: Voltar uma página.
   - `Alt + /`: Ir para a linha específica.

3. **Edição de Texto:**
   - `Ctrl + K`: Cortar a linha atual.
   - `Ctrl + U`: Colar o texto cortado.
   - `Alt + A`: Alternar entre inserção e sobrescrita.
   - `Alt + ^` (Alt + 6): Marcar o início de uma seleção.
   - `Alt + \`: Marcar o fim de uma seleção.

4. **Busca e Substituição:**
   - `Ctrl + W`: Buscar texto.
   - `Ctrl + \`: Buscar e substituir texto.

5. **Outros Comandos Úteis:**
   - `Ctrl + G`: Exibir a ajuda do Nano.
   - `Ctrl + C`: Mostrar a posição atual do cursor.
